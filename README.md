
# Rock Paper Scissors Game

A lightweight and interactive **Rock Paper Scissors** game built using **vanilla JavaScript**, focusing on clean UI, simple game logic, and beginner-friendly code structure.

This project demonstrates core front-end concepts such as DOM manipulation, event handling, and conditional logic.

---

## Overview

The game allows a user to play Rock, Paper, or Scissors against the computer.
The computer’s move is randomly generated, and the result of each round is displayed in real time with updated scores.

---

## Features

* Interactive click-based gameplay
* Randomized computer choice logic
* Real-time score tracking
* Clear win, lose, and draw states
* Clean and minimal user interface
* Fully responsive layout

---

## Technologies Used

* **HTML5** – Markup and structure
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Game logic and interactivity

## Game Rules

* Rock defeats Scissors
* Scissors defeats Paper
* Paper defeats Rock
* Identical choices result in a draw
